Lesson 04

繰り返し処理（Loops）

同じ処理を何度も実行する場合や、データの集まり（配列）を一つずつ取り出す場合に欠かせない仕組みです。

1 foreach 文（配列の反復処理）

配列の各要素を順番に取り出して処理します。bingo.php で、生成した5x5のカードデータを表示する際に使用しています。

// 配列の中身を一つずつ $row に代入してループ
foreach ($rows as $row) {
    foreach ($row as $value) {
        // $value を表示する処理
        echo $value;
    }
}

2 while 文（条件による反復）

指定した条件が「真」である限り、処理を繰り返します。calculate_loan.php では、ローン残高が0円になるまで計算を続けています。

// 残高がある間、繰り返し計算を行う
while ($loan > 0) {
    $loan -= $payment;
    // ...
}
Important: while 文は、条件がいつまでも「真」のままだと、無限ループに陥ってプログラムが止まらなくなります。必ず終了条件へ近づく処理を含めましょう。

3 for 文（回数指定による反復）

「10回繰り返す」など、回数が決まっている場合に便利です。bingo.php で縦列を横列に変換するデータ処理に使用しています。

// カウンタ変数が5未満の間、繰り返す
for ($i = 0; $i < 5; $i++) {
    // 5回実行される処理
}

"ループと配列を組み合わせることで、大量のデータを一瞬で処理できるようになります。"
