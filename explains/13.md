## はじめに

PHPで日付や時刻を扱う方法は、伝統的な関数ベースの書き方と、モダンなオブジェクト指向ベースの書き方の2種類があります。それぞれの特徴と、実用的なカレンダーの実装方法を解説します。

## date() 関数とタイムスタンプ

**タイムスタンプ**とは、1970年1月1日 00:00:00（UTC）からの経過秒数のことで、`time()` 関数で現在のタイムスタンプを取得できます。`date()` 関数にフォーマット文字列とタイムスタンプを渡すと、指定した形式の日付文字列が得られます。

```php
<?php
date_default_timezone_set('Asia/Tokyo');

// 現在のタイムスタンプを取得
$now_time = time();

// x日後のタイムスタンプ（秒単位で計算）
$day = 7;
$next_time = $now_time + (24 * 60 * 60) * $day;

// date() でフォーマットして表示
$year         = date('Y');           // 4桁の年
$month        = date('m');           // 2桁の月（01〜12）
$days         = date('t');           // 今月の日数
$today_string = date('Y/m/d H:i:s'); // 2026/03/02 15:30:00
```

主なフォーマット文字は次のとおりです。

| 書式文字 | 意味 | 出力例 |
| :--- | :--- | :--- |
| `Y` | 4桁の年 | 2026 |
| `m` | 2桁の月（ゼロ埋め） | 03 |
| `d` | 2桁の日（ゼロ埋め） | 02 |
| `H` | 24時間表記の時（ゼロ埋め） | 15 |
| `i` | 分（ゼロ埋め） | 30 |
| `s` | 秒（ゼロ埋め） | 00 |
| `t` | 指定した月の日数 | 28 |
| `w` | 曜日の数値（0:日 〜 6:土） | 0 |
| `n` | 月（ゼロ埋めなし） | 3 |

> **タイムゾーンの設定**
> PHPで日付を扱う前に `date_default_timezone_set('Asia/Tokyo');` でタイムゾーンを設定しないと、サーバーの設定によって時刻がずれることがあります。必ずスクリプトの先頭で呼び出しましょう。

## strtotime() による相対指定

`strtotime()` は「1日後」「3ヶ月前」「次の日曜日」といった人間が理解しやすい**英語表現**をタイムスタンプに変換してくれる関数です。`date()` と組み合わせることで、相対的な日付を柔軟に計算できます。

```php
<?php
// 1日後のタイムスタンプ → 日付文字列に変換
$next_day_time = strtotime('+1 day');
$next_day      = date('Y/m/d', $next_day_time);

// 3ヶ月前
$prev_day_time = strtotime('-3 month');
$prev_day      = date('Y/m/d', $prev_day_time);

// 3時間後
$next_hour_time = strtotime('+3 hour');
$next_hour      = date('Y/m/d H:i:s', $next_hour_time);

// 次の日曜日
$next_week_time = strtotime('+1 sunday');
$next_week      = date('Y/m/d H:i:s', $next_week_time);
```

主な英語表現の例：`+1 day`（1日後）、`-3 month`（3ヶ月前）、`+3 hour`（3時間後）、`+1 sunday`（次の日曜日）、`first day of last month`（先月の初日）など、直感的な英語が使えます。

## DateTime クラス（モダンな手法）

モダンなPHP開発では、`DateTime` クラスを使うのが一般的です。`setDate()` や `modify()` などのメソッドで直感的に日付を操作でき、オブジェクト同士を比較演算子でそのまま比較できます。

```php
<?php
date_default_timezone_set('Asia/Tokyo');

// インスタンス作成（引数なしで現在日時）
$date = new DateTime();

// 特定の日時をセット（メソッドチェーン）
$date->setDate(2022, 3, 10)->setTime(10, 30, 45);

// フォーマットして文字列化
$date_string = $date->format('Y-m-d H:i:s'); // 2022-03-10 10:30:45

// modify() で相対的に変更
$date2 = new DateTime();
$date2->setDate(2022, 3, 10)->setTime(10, 30, 45);
$date2->modify('+1 day');
$date_string2 = $date2->format('Y-m-d H:i:s'); // 2022-03-11 10:30:45

// 日付の比較（そのまま比較演算子が使える）
$is_match = ($date < $date2); // true（$date の方が前の日時）
```

主なメソッドは次のとおりです。

| メソッド | 意味 | 例 |
| :--- | :--- | :--- |
| `setDate(Y, m, d)` | 年・月・日を設定する | `setDate(2022, 3, 10)` |
| `setTime(H, i, s)` | 時・分・秒を設定する | `setTime(10, 30, 45)` |
| `modify(string)` | 相対的な日時変更（strtotime 書式） | `modify('+1 month')` |
| `format(string)` | フォーマット文字列に変換する | `format('Y-m-d')` |
| `clone $date` | オブジェクトのコピーを作る | `$copy = clone $date` |

> **clone に注意**
> PHPのオブジェクトは代入しても**参照コピー**になります。別の変数として独立させたい場合は `clone` キーワードを使います。`calendar.php` の前月・次月リンク生成でも `clone $today` が使われています。

## 応用：カレンダーの生成ロジック

カレンダーを作る際は、**2次元配列**に日付オブジェクトをマッピングしていく考え方が基本です。`calendar.php` では以下のロジックで実装しています。

```php
<?php
date_default_timezone_set('Asia/Tokyo');

$today = new DateTime();
$year  = $today->format('Y');
$month = $today->format('n'); // ゼロ埋めなしの月

// クエリパラメータで月を切り替え可能に
if (isset($_GET['y'])) $year  = (int)$_GET['y'];
if (isset($_GET['m'])) $month = (int)$_GET['m'];

$today->setDate($year, $month, 1);
$end_day = (int)$today->format('t'); // 月の日数
$days    = range(1, $end_day);

// カレンダー配列の初期化（最大6行 × 7列、null で埋める）
$calendar = array_fill(0, 6, array_fill(0, 7, null));
$row = 0;

foreach ($days as $day) {
    $date        = new DateTime();
    $date->setDate($year, $month, $day);
    $week_number = (int)$date->format('w'); // 0:日 〜 6:土

    $calendar[$row][$week_number] = $date;

    // 土曜日（6）に達したら次の行へ
    if ($week_number === 6) {
        $row++;
    }
}
```

実装の手順は次のとおりです。

1. **月初と末日の特定** — `setDate($year, $month, 1)` で1日にセットし、`format('t')` でその月が何日まであるかを取得します。
2. **曜日ごとの列にセット** — `format('w')` で曜日の数値（0:日 〜 6:土）を取得し、`$calendar[$row][$week_number] = $date` に格納します。
3. **土曜日で行を切り替え** — `$week_number === 6`（土曜日）のタイミングで `$row++` して次の週（行）に進みます。
4. **2次元配列を HTML に展開** — 入れ子の `foreach` でループし、CSS Grid（`grid-template-columns: repeat(7, 1fr)`）で7列のカレンダーを描画します。

> **空行のスキップ**
> 最大6行で初期化していますが、月によっては5行で収まります。`array_filter($row_date)` で全セルが `null` の行をスキップすることで、余白のない綺麗なカレンダーになります。

## まとめ

- **`time()`** — 現在の Unix タイムスタンプ（1970年1月1日からの経過秒数）を返す。
- **`date(書式, タイムスタンプ)`** — タイムスタンプをフォーマット文字列に変換する。
- **`strtotime()`** — 英語表現（`+1 day` など）をタイムスタンプに変換する。
- **`DateTime` クラス** — `setDate()` / `setTime()` / `modify()` / `format()` でオブジェクト指向に日付を操作できる。
- **比較演算子** — `DateTime` オブジェクト同士は `<` `>` などの比較演算子でそのまま比較できる。
- **`clone`** — オブジェクトの参照コピーを防ぎ、独立したコピーを作るために使う。
- **2次元配列** — カレンダーは `$calendar[$行][$曜日]` の形で日付オブジェクトをマッピングして生成する。
- **`date_default_timezone_set('Asia/Tokyo')`** — 日付処理を行うファイルの先頭で必ず設定する。
