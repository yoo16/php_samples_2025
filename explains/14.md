## オブジェクト指向とは

### オブジェクトが基本

ゲームで登場する「カード」を1つの実体として考えます。
このカードには「名前」「攻撃力」「HP」などの**特徴**があり、
「攻撃する」「レベルアップする」などの**動作**があります。

このように、

> **特徴（データ）＋動作（処理）をまとめたものがオブジェクト**

です。

今回の例では、カードそのものがオブジェクトになります。

---

### オブジェクトで役割分担する

カードバトルゲームを考えてみましょう。
まずプログラミングをする前に、カードバトルゲームの役割を考えてみましょう。

| オブジェクト | 役割 |
| ------ | ------------ |
| カード | 戦闘・成長・スキルを持つ |
| プレイヤー | カードを操作する |
| バトル | ダメージ計算や勝敗判定 |
| ゲーム | ゲームの流れを管理する |
| UI | ゲーム画面を表示する |

オブジェクト指向では、これらの役割をオブジェクトとして表現します。

## プログラムの構成

今回のサンプルは3つのファイルで構成されています。

| ファイル | 内容 |
| --------------- | ---------------------------------- |
| `instance.php` | `BaseCard` から直接インスタンスを生成して表示する |
| `card_list.php` | 4種類のカードを一覧表示する |
| `card_select.php` | カードを選択し、経験値獲得・レベルアップを試す |

クラスファイルは `models/` フォルダに分けて管理しています。

```treeview
./
├── models/
│   ├── BaseCard.php     ← 全カード共通の設計図（親クラス）
│   ├── AquaCard.php     ← 水属性カード（子クラス）
│   ├── ForestCard.php   ← 風属性カード（子クラス）
│   ├── KnightCard.php   ← 火属性カード（子クラス）
│   └── ThunderCard.php  ← 雷属性カード（子クラス）
├── instance.php
├── card_list.php
└── card_select.php
```

## クラスとは

クラスとは **オブジェクトの設計図**で、実際にオブジェクトを作るためのひな形です。
今回のプログラムでは、**カードの基本設計**を担当する `BaseCard` がクラスです。

### クラス定義
クラスを定義するには、`class` キーワードを用いてクラス名 `BaseCard` を定義します。

```php
class BaseCard
{
    // ここにプロパティやメソッドを書く
}
```

### プロパティ（メンバ変数）

クラスには、そのクラスの特徴となるデータを定義します。
これらは **プロパティ（メンバ変数）** と呼ばれます。

| 項目 | 意味 |
| ----------------- | ----------- |
| $level | レベル |
| $name | 名前 |
| $attack | 攻撃力 |
| $defense | 防御力 |
| $hp / $maxHp | 現在HP / 最大HP |
| $mp / $maxMp | 現在MP / 最大MP |
| $exp | 経験値 |
| $element | 属性 |
| $image | 画像パス |
| $specialSkill | 必殺技名 |
| $specialSkillPower | 必殺技威力 |

```php
class BaseCard
{
    // プロパティ
    public int $level = 1;
    public string $name = '';
    public int $attack = 0;
    public int $defense = 0;
    public int $hp = 0;
    public int $maxHp = 0;
    public int $mp = 0;
    public int $maxMp = 0;
    public int $exp = 0;
    public string $element = '';
    public string $image = '';
    public string $specialSkill = '';
    public int $specialSkillPower = 0;
}
```

## インスタンス

クラス（設計図）から作られた実体を **インスタンス** といいます。
`new` キーワードを使って、設計図から具体的なオブジェクトを生成します。

`instance.php` では、次のように `BaseCard` クラスからインスタンスを作成しています。

```php
// BaseCard クラスからインスタンスを生成
$card = new BaseCard(
    '水の精霊',      // 名前
    18,             // 攻撃力
    25,             // 防御力
    100,            // HP
    3,              // MP
    '水',           // 属性
    'AquaCard.png', // 画像
    'ハイドロポンプ', // 必殺技
    35              // 必殺技威力
);
```

この `$card` が、メモリ上に実体として作られた「カードオブジェクト」です。
プロパティへのアクセスは `->` を使います。

```php
echo $card->name;   // 水の精霊
echo $card->attack; // 18
echo $card->hp;     // 100
```

## クラスメソッド

クラスには、そのクラスの動作（機能）を定義します。
これらは **クラスメソッド** と呼ばれます。


| メソッド | 役割 |
| ---------------- | -------- |
| `showStatus()` | ステータス表示 |
| `attack($target)` | 通常攻撃（相手にダメージを与える） |
| `specialSkill($target)` | 必殺技 |
| `gainExp($exp)` | 経験値取得 |
| `isLevelUp()` | レベルアップ判定 |
| `levelUp()` | レベルアップ処理 |

### $this の使い方
メソッドの中で自分自身のプロパティにアクセスするときは、`$this->プロパティ名` と書きます。

```php
public function gainExp(int $exp): void
{
    // 自分の exp プロパティに引数の $exp を加算する
    $this->exp += $exp;
}
```

### attack() メソッド

引数に攻撃対象のカードを受け取り、ダメージを計算して相手の HP を減らします。
返り値は実際に与えたダメージ量です。

```php
public function attack(BaseCard $target): int
{
    // ダメージ計算: (攻撃力 * 1.5 - 相手の防御力) + 乱数
    $baseDmg = ($this->attack * 1.5) - $target->defense;
    $random = rand(-5, 5);
    $dmg = (int)($baseDmg + $random);

    // 最低ダメージ保証 (攻撃力の 20%)
    $minDmg = (int)($this->attack * 0.2);
    if ($dmg < $minDmg) $dmg = $minDmg;

    $target->hp -= $dmg;
    if ($target->hp < 0) $target->hp = 0;

    return $dmg;
}
```

### specialSkill() メソッド

MP を1消費して必殺技を発動します。通常攻撃より威力が高いですが、MP が 0 の場合は発動できません。

```php
public function specialSkill(BaseCard $target): int
{
    if ($this->mp <= 0) return 0; // MP不足なら発動しない
    $this->mp--;

    // スキルダメージ: (スキル威力 * 1.2 - 相手の防御力 * 0.5)
    $baseDmg = ($this->specialSkillPower * 1.2) - ($target->defense * 0.5);
    $random = rand(-10, 10);
    $dmg = (int)($baseDmg + $random);

    if ($dmg < 0) $dmg = 0;
    $target->hp -= $dmg;
    if ($target->hp < 0) $target->hp = 0;

    return $dmg;
}
```

### isLevelUp() / levelUp() メソッド

経験値が閾値を超えていればレベルアップします。レベルアップ時は各ステータスが上昇し、HP/MP が全回復します。

```php
public function isLevelUp(): bool
{
    if ($this->level >= 10) return false;
    return $this->exp >= $this->levelUpThresholds[$this->level - 1];
}

public function levelUp(): void
{
    $this->level++;
    $this->attack  += 5;
    $this->defense += 2;
    $this->maxHp   += 20;
    $this->hp       = $this->maxHp; // レベルアップで全回復
    $this->maxMp   += 1;
    $this->mp       = $this->maxMp;
}
```

## コンストラクタとは？

```php
public function __construct(...)
```

これは、インスタンス生成時（new した時）に**自動実行される特別なメソッド**です。
主にプロパティの初期化に使われます。

### 親クラス（BaseCard）のコンストラクタ
共通の初期化処理を書いておきます。

```php
public function __construct(string $name, int $attack, ...) {
    $this->name = $name;
    $this->attack = $attack;
    // ...
}
```

## 継承 (extends)

**継承**とは、あるクラスの機能を引き継いで、新しいクラスを作ることです。

* **親クラス (BaseCard)**: 全てのカードに共通する機能（攻撃、レベルアップなど）を持つ。
* **子クラス (AquaCard, ForestCard など)**: 親の機能を引き継ぎつつ、個別の名前やステータスを持つ。

```php
class AquaCard extends BaseCard // BaseCard を継承
```

これによって、`AquaCard` は自分で `attack()` メソッドを書かなくても、`BaseCard` が持っている `attack()` を使うことができます。

### 子クラス（AquaCard）のコンストラクタ
`parent::__construct()` を使って、親クラスのコンストラクタを呼び出します。これにより、共通の初期化処理を再利用できます。

```php
class AquaCard extends BaseCard {
    public function __construct() {
        // 親クラスのコンストラクタに具体的な値を渡して呼び出す
        parent::__construct('水の精霊', 18, 25, 100, 3, '水', 'AquaCard.png', 'ハイドロポンプ', 35);
    }
}
```

4種類のカードは、それぞれ異なるステータスで初期化されます。

| クラス | 名前 | 攻撃 | 防御 | 属性 | 必殺技 |
| ------------ | -------- | -- | -- | -- | ------------- |
| AquaCard | 水の精霊 | 18 | 25 | 水 | ハイドロポンプ |
| ForestCard | 森の守護者 | 22 | 20 | 風 | リーフストーム |
| KnightCard | 炎の騎士 | 25 | 15 | 火 | ブレイズソード |
| ThunderCard | 雷の召喚獣 | 30 | 10 | 雷 | ボルテッカー |

## セッションとインスタンス

`card_select.php` では、ページをまたいでカードの状態を保持するために **セッション** を使っています。

PHPはリクエストごとに変数が初期化されるため、経験値を加算してもページを移動すると元に戻ってしまいます。
セッションにインスタンスを保存することで、状態を持続させています。

```php
session_start();

// セッションにカードが保存されていればそれを使う
$card = $_SESSION['cards'][$card_id] ?? null;
if (!$card) $card = $cards[$card_id]; // なければ新規生成

// 経験値を加算してレベルアップ判定
if (isset($_GET['exp'])) {
    $card->gainExp($_GET['exp']);
    if ($card->isLevelUp()) $card->levelUp();

    // 更新した状態をセッションに保存
    $_SESSION['cards'][$card_id] = $card;
}
```

処理の流れ：

1. `session_start()` でセッションを開始
2. セッションに保存済みのカードがあれば取り出す
3. 経験値を加算 → レベルアップ判定
4. 更新したインスタンスをセッションに上書き保存

---

## まとめ

* **クラス** = 設計図（class）
* **インスタンス** = 設計図から作られた実体（new）
* **プロパティ** = オブジェクトが持つデータ（変数）
* **メソッド** = オブジェクトができること（関数）
* **コンストラクタ** = `new` したときに自動実行される初期化処理（`__construct`）
* **継承 (extends)** = 他のクラスの機能を引き継ぐ
* **セッション** = ページをまたいでデータ（インスタンス）を保存する仕組み
