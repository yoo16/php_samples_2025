# 教材 md 作成

新たな教材コンテンツ md を作成する。

## 参考プログラム

- 指定したフォルダのプログラム
- explaination.php のプログラム内容は除く

## 参考ドキュメント

- docs/web_basic/php_basic/
- 指定した番号の md
- explaination.php の内容を参考に

## 修正内容

- わかりやすい内容に修正
- 誤字脱字チェック
- 重要キーワードは **太字**
- 専門用語は `コード基語`
- 引用は `>`
- PHPコードは ```php ...```
- HTMLコードは ```html ...```
- CSSコードは ```css ...```
- JavaScriptコードは ```javascript ...```
- 可能であれば svg も作成
